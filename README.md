# calculator-project

The .env has been uploaded to github as it has the database URI, which is used to automatically connect to the database. </br></br>


How to run the project (Ensure that node.js and git are downloaded on your computer): </br>

1- Clone the repo to your computer (git clone 'the url from github') </br>
2- cd calculator-project </br>
3- npm i </br>
4- npm start </br>
5- Open web browser at localhost:5173 </br>
6- To run the jest unit test: kill the server(if running) and then type npm run test
